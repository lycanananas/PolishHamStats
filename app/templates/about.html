{% translate "znakisp.pl" %}

{% blocktranslate trimmed %} znakisp.pl to projekt agregujący publiczne dane o polskich znakach i pozwoleniach krótkofalarskich. Serwis pozwala szybko sprawdzić status znaku, historię widoczności oraz podstawowe informacje o pozwoleniach. {% endblocktranslate %}

{% blocktranslate trimmed %} Dane UKE są synchronizowane po stronie serwera, a dane PZK pobierane na żądanie i cache'owane per znak. {% endblocktranslate %}

{% translate "Aktualność i wiarygodność danych" %}

{% blocktranslate trimmed %} Projekt regularnie odświeża bazę danych UKE, dzięki czemu zmiany w znakach i pozwoleniach są systematycznie uwzględniane. Oznacza to, że serwis pokazuje możliwie aktualny obraz sytuacji, a jednocześnie zachowuje historię zmian, aby łatwiej było zrozumieć kontekst. {% endblocktranslate %}

{% blocktranslate trimmed %} Dane z PZK są pobierane dla konkretnego znaku wtedy, gdy użytkownik ich potrzebuje. Taki model zmniejsza ryzyko pokazywania przypadkowych lub niepotrzebnych informacji i pomaga utrzymać spójność prezentowanych wyników. {% endblocktranslate %}

{% blocktranslate trimmed %} W praktyce serwis ma pełnić rolę bezpiecznego, pomocniczego źródła informacji do codziennej pracy i weryfikacji, a nie zastępować oficjalne rejestry publikowane przez instytucje źródłowe. {% endblocktranslate %}

{% translate "Prywatność" %}

{% blocktranslate trimmed %} Projekt nie zbiera dodatkowych danych osobowych użytkowników strony ponad standardowe dane techniczne wymagane do działania serwisu. Prezentowane informacje pochodzą wyłącznie z danych publicznie dostępnych na stronach UKE oraz PZK. {% endblocktranslate %}
